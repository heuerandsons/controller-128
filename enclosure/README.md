# enclosure

### Features

External clock only

  * Clock in
  * Reset in
  * 8x Trigger out
  * 128x grid LED buttons
  * USB 5v power

Internal clock only

  * Clock out
  * 7x Trigger out
  * Tempo potentiometer
  * Swing potentiometer
  * Humanize potentiometer
  * 128x grid LED buttons
  * USB 5v power

Internal and external clock

  * Clock in
  * Reset in
  * Clock out
  * 8x Trigger out
  * Tempo potentiometer
  * Swing potentiometer
  * Humanize potentiometer
  * 128x grid LED buttons
  * USB 5v power

Other ideas

  * LCD screen
  * Rotary encoder(s)
  * Transport buttons
  * In/out indicator LEDs
